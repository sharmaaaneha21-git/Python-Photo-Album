# Python Photo Slideshow Album

This project is a simple Photo Slideshow application built using Python, Tkinter, and Pillow.

The application displays a slideshow of images when the user clicks the "Play the Slideshow" button.

## Features

- Display images in a slideshow
- Automatic image switching every 2 seconds
- Simple GUI using Tkinter
- Image processing using Pillow

## Technologies Used

- Python
- Tkinter
- Pillow (PIL)

## How It Works

1. The program loads image paths from a list.
2. Each image is opened and resized using Pillow.
3. Images are converted into a Tkinter-compatible format.
4. When the button is clicked, the slideshow starts.
5. Each image is displayed for 2 seconds.

## How to Run

1. Install Pillow

pip install pillow

2. Run the program

python main.py
